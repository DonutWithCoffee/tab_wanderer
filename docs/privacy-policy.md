# Политика конфиденциальности Tab Wanderer

Дата вступления в силу: 2026-07-05

## Назначение расширения

Tab Wanderer — расширение Chrome для внутреннего использования авторизованными пользователями Amperkot admin.

Расширение предназначено для локального мониторинга заказов в административной панели Amperkot, показа уведомлений об изменениях, ведения списка отслеживаемых заказов, напоминаний и вспомогательной проверки/привязки штрихкодов между warehouse/Ozon страницами.

## Какие данные обрабатываются

Расширение может обрабатывать содержимое страниц Amperkot admin, warehouse pages и seller.ozon.ru, доступных авторизованному пользователю.

Это может включать:

- номера заказов;
- статусы заказов;
- данные о доставке;
- данные об оплате;
- контрагента;
- телефон или другие детали заказа, если они присутствуют на странице;
- товарные идентификаторы;
- штрихкоды и связанные данные warehouse/Ozon flow.

## Как используются данные

Данные используются только для основных функций расширения:

- локальный мониторинг заказов;
- уведомления о новых и изменённых заказах;
- отслеживаемые заказы;
- напоминания;
- диагностический журнал для поддержки;
- warehouse/Ozon barcode helper.

## Где хранятся данные

Данные хранятся локально в браузере пользователя через Chrome extension storage.

## Передача данных

Расширение не продаёт пользовательские данные.

Расширение не передаёт данные заказов на сторонние серверы расширения.

Расширение не использует пользовательские данные для рекламы, кредитования, определения платёжеспособности или целей, не связанных с основными функциями расширения.

## Удалённый код

Расширение не загружает и не выполняет удалённый JavaScript-код.

Исполняемый код расширения поставляется внутри пакета расширения.

## Доступ к сайтам

Расширение работает только на доменах, указанных в manifest:

- amperkot.ru;
- seller.ozon.ru.

Доступ используется только в рамках страниц, доступных авторизованному пользователю, и только для функций расширения.

## Контакты

По вопросам конфиденциальности обратитесь к администратору Amperkot или владельцу расширения.
