# tab_wanderer

Расширение Chrome для отслеживания изменений заказов в админке и отправки уведомлений.

---

## 📌 Назначение

Плагин автоматически отслеживает:

* новые заказы
* изменения существующих заказов
* смену статусов

И уведомляет пользователя в реальном времени.

---

## ⚙️ Как это работает

1. Создаётся служебная вкладка (worker)
2. Формируется URL на основе monitorScope
3. Плагин собирает список заказов
4. Формируется хэш состояния заказа
5. Определяются изменения
6. Применяются правила уведомлений
7. Отправляется уведомление

---

## 🔔 Уведомления

Уведомления приходят при:

* появлении нового заказа
* изменении заказа

При клике открывается карточка заказа.

---

## 🏷 Метки в уведомлениях

Некоторые заказы помечаются:

* **(ОЗОН)** — если контрагент `OZON (ОЗОН)`
* **(Юрик)** — если тип оплаты `Безналичный расчет для юридических лиц`

---

## ⚙️ Настройки

В popup доступны:

* **START / STOP**
* фильтры наблюдения (monitor scope):

  * статус (ID)
  * доставка (ID)
  * оплата (ID)

* чекбоксы:

  * Игнорировать ОЗОН
  * Игнорировать юриков

Фильтры формируют URL запрос к админке и определяют,
какие заказы участвуют в мониторинге.

---

## 🔄 Поведение системы

### Baseline

Чтобы избежать ложных уведомлений:

* при запуске
* при изменении настроек

плагин пересобирает текущее состояние заказов.

Старые заказы не считаются новыми.

---

## 🧠 Архитектура (упрощённо)

Плагин состоит из:

* **Core** — стабильная логика (worker, baseline, обработка)
* **Rules** — правила уведомлений (decision layer)
* **Config/UI** — пользовательские настройки
* **Monitor Scope** — фильтрация заказов через URL (status/delivery/payment)
