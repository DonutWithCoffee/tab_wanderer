# Tab Wanderer

Chrome extension для отслеживания заказов в админке amperkot.ru

---

## 🚀 Версия

**0.9.1-dev**

---

## 🎯 Цель

Отслеживание изменений заказов с уведомлениями:

- новые заказы
- изменение статуса
- изменение оплаты / доставки

Без влияния на другие вкладки.

---

## ⚙️ Архитектура

### 🔴 Основа

- DOM parsing
- обновление через reload

---

### 🟢 Worker Tab

- создаётся 1 скрытая вкладка
- pinned + inactive
- используется только она
- остальные вкладки игнорируются

---

### 🟡 Обновление

- каждые 30 секунд
- только worker tab

---

### 🧠 Storage (chrome.storage.local)

- `ordersDB` — заказы
- `ordersHashDB` — хэши
- `workerTabId` — id вкладки
- `lastBaselineDate` — дата baseline

---

### 🧠 Hash система

```
hash = id + status + delivery + payment
```

- используется normalize()
- защита от дублей
- триггер изменений

---

### 🧠 Baseline

- первый snapshot = baseline
- без уведомлений
- авто baseline после 09:00
- ручной baseline (popup)
- reset baseline

---

### 🔔 Уведомления

Триггер:

- изменение hash

Содержат:

- номер заказа
- статус
- доставка
- оплата

---

### 🧪 Тесты

Через popup:

- отправка тестовых заказов
- не используют DOM
- `isTest: true`
- не изменяют базу

---

### 🧱 Парсинг

- через заголовки таблицы (`thead`)
- не зависит от порядка колонок
- устойчив к изменениям DOM

---
