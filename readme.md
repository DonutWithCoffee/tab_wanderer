# tab_wanderer

Расширение Chrome для отслеживания изменений заказов в админке и отправки уведомлений.

---

## 📌 Назначение

Плагин автоматически отслеживает:

* новые заказы
* изменения существующих заказов
* смену статусов

И уведомляет пользователя в реальном времени.

---

## ⚙️ Как это работает

1. Создаётся служебная вкладка (worker)
2. Плагин собирает список заказов
3. Формируется хэш состояния заказа
4. Определяются изменения
5. Применяются правила уведомлений
6. Отправляется уведомление

---

## 🔔 Уведомления

Уведомления приходят при:

* появлении нового заказа
* изменении заказа

При клике открывается карточка заказа.

---

## 🏷 Метки в уведомлениях

Некоторые заказы помечаются:

* **(ОЗОН)** — если контрагент `OZON (ОЗОН)`
* **(Юрик)** — если тип оплаты `Безналичный расчет для юридических лиц`

---

## ⚙️ Настройки

В popup доступны:

* **START / STOP**
* чекбоксы:

  * Игнорировать ОЗОН
  * Игнорировать юриков

Изменения применяются сразу.

---

## 🔄 Поведение системы

### Baseline

Чтобы избежать ложных уведомлений:

* при запуске
* при изменении настроек

плагин пересобирает текущее состояние заказов.

Старые заказы не считаются новыми.

---

## 🧠 Архитектура (упрощённо)

Плагин состоит из:

* **Core** — стабильная логика работы
* **Rules** — правила уведомлений
* **Config/UI** — настройки пользователя
