# tab_wanderer

Chrome extension для автоматического отслеживания заказов в админке amperkot.ru.

---

## 🚀 Статус

- stable core: `v0.9.2`
- development: `0.9.3-dev`

---

## 🎯 Что делает

- отслеживает изменения заказов
- отправляет уведомления без участия пользователя
- работает через отдельную worker-вкладку

---

## ⚙️ Как работает

1. открывается служебная вкладка (worker)
2. собираются заказы
3. изменения определяются через хэш
4. перед уведомлением применяется rules engine
5. отправляется уведомление или пропуск

---

## 🔔 Уведомления

Уведомления проходят через правила:

- фильтрация по статусу
- фильтрация по доставке
- фильтрация по оплате

Цель текущей версии — убрать ложные уведомления и сделать поведение предсказуемым.

---

## 🧱 Ограничения

- не переиспользует чужие вкладки
- не ищет вкладки по URL
- не зависит от UI страницы
- минимальный UI (только START / STOP)

---

## 🗺️ Roadmap

- улучшение правил уведомлений
- разделение типов событий (новый / изменение / статус)
- настройка уведомлений пользователем
- переход в заказ по клику на уведомление (v1.0)
