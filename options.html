Настройки tab_wanderer

Эта страница будет использоваться для расширенных настроек мониторинга и уведомлений.

Назначение страницы

Popup останется быстрым контролом START / STOP и краткой сводки состояния. Расширенные настройки будут перенесены сюда.

Что будет здесь

  • Режим мониторинга
  • Область мониторинга
  • Notification triggers
  • Поля изменений
  • Apply / Reset

Режим мониторинга

Windowed отслеживает первую страницу и deep sync. Active ограничивается первой страницей.
